# XCSS Language Support for VS Code

![Version](https://img.shields.io/badge/version-0.0.1-blue.svg)
![VS Code](https://img.shields.io/badge/VS%20Code-1.86.0+-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Soporte oficial para el lenguaje **XCSS** (eXtended CSS) en Visual Studio Code.

## 📦 Características

- ✅ **Resaltado de sintaxis** completo para XCSS
- ✅ **Auto-cierre** de llaves, corchetes y paréntesis
- ✅ **Comentarios** de línea (`//`) y bloque (`/* */`)
- ✅ **Sangrado automático** para bloques de código
- ✅ **Reconocimiento** de variables, funciones, mixins y palabras clave
- ✅ **Soporte** para propiedades y selectores CSS

## 🚀 Instalación

### Desde el Marketplace
1. Abre VS Code
2. Ve a la pestaña de Extensiones (`Ctrl+Shift+X`)
3. Busca `XCSS Language Support`
4. Haz clic en **Instalar**

### Desde un archivo VSIX
```bash
code --install-extension xcss-0.0.1.vsix
